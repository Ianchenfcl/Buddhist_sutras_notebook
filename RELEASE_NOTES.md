# Release v0.7.4 - 修復語音導師 (Voice Mode) 知識庫認知盲區

此版本修復了語音導師會誤以為自己只有 8 個章節知識，而拒絕回答其他章節（如《金剛經》第二章）的問題。

## 🌟 核心更新 (Key Updates)

1. **解除語音導師的視野限制 (Voice Assistant Prompt Fix)**：
   - 先前為了讓語音模型知道當前有哪些文獻，後端會隨機抓取 8 篇文獻標題塞入系統提示詞中。這導致強大的 Gemini 模型被「畫地自限」，以為自己只能回答那 8 個章節。
   - 本次更新已將系統提示詞改寫，直接告訴導師：「您擁有完整的《金剛經》全 32 品，請盡情運用您內建的佛法大智慧來開示。」
   - 現在您不管問第幾章，語音導師都能對答如流了！

# Release v0.7.3 - 開箱即用：預載入知識庫資料庫

此版本解決了首次下載的用戶（如您的同事）打開程式時會發現「參考文件為空」的問題。

## 🌟 核心更新 (Key Updates)

1. **內建預先嵌入的向量資料庫 (Pre-seeded Vector Database)**：
   - 之前為了節省版本庫大小，我們在 `.gitignore` 中隱藏了 `.db` 與 `chroma_db`。這導致新使用者下載原始碼後，必須手動執行 `import_sutras.py` 才能建立知識庫（且需要設定 API Key）。
   - 本次更新已將高達 150MB 的**完整金剛經向量資料庫 (ChromaDB) 與系統關聯庫 (SQLite)** 正式打包進專案中。
   - **開箱即用**：您的同事現在只要下載這個最新版 Release，解壓縮後雙擊 `start.bat`，不需設定任何 API Key 就能立刻看到左側滿滿的 32 份《金剛經》參考文件，並能直接使用搜尋與導讀功能！

# Release v0.7.0 - 佛AI (Dharma AI) 誕生

這是佛AI (Dharma AI) 的首個正式里程碑版本，宣告本專案正式由前身的社交知識庫轉型為**純粹、高質感的佛學與經典參悟智慧助理**。

## 🌟 核心更新 (Key Updates)

1. **全面佛學化大整改**：
   - 所有的前端介面標誌、歡迎詞與預設問候按鈕，皆已全面替換為《金剛經》與大乘佛法的核心精神。
   - 包含中英文雙語的提示詞與翻譯，都已改寫為「佛學大師 (Buddhist Master)」的情境，提供客觀、充滿禪意的對話體驗。
2. **語音導師 (Voice AI Tutor) 升級**：
   - 即時語音通話的背後系統提示詞已轉化為具備「應無所住」般若智慧的佛學導師，提供溫暖、口語化且精準的佛學解惑。
3. **測驗系統精簡 (Quiz System Cleanup)**：
   - 移除了原先殘留的非佛學相關測驗，保留並重構了以《金剛經》核心概念為主的**「佛理參悟測驗」**，專注於空性、四相等核心教義的測驗。
4. **README 與文檔更新**：
   - 全新編寫了中英雙語的 `README.md`，並將「快速起手 (Quick Start)」步驟移至更顯眼的位置，方便新手無痛安裝並進入佛學知識庫。

## 🛠 技術修正 (Technical Fixes)
- 修正了資料庫 (SQLite/ChromaDB) 啟動時的初始化邏輯，使得預設筆記本名稱「金剛經」能被永久保存，不再於伺服器重啟時被覆蓋。
- 優化了文檔清單的排序演算法，現已支援以中文數字（第一分、第二分）進行正確的章節排序。
- 將 `top_k` 從 7 提升至 15，搭配章節標題前綴寫入 Chunk 的策略，大幅強化了 RAG 系統面對「模糊搜尋（如：詢問特定章節名）」時的召回精準度。
